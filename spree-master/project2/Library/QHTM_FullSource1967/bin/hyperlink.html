Links

Links, as you would expect, you can have links to web sites outside of your aplication. Such as this one. Of course images work as links too.

Link tips

Hyperlinks get tooltips displayed which by default is the URL. You can override this by specifying a title="". For example this link will display the HTML message box in action but the link is internal and is quite ugly. Hover the mouse cursor over the link to see what happens. Yup, you can even have HTML in the tooltips!
