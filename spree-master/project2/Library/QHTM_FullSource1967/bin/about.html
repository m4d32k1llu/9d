QHTM Development Stuff

QHTM is written entirely in C++. It has an extensible hierarchy of classes that mean adding new object capabilities is fairly simple.

QHTM is tuned using Numega's TrueTime profiler and it is run through Numega's BoundsChecker prior to being released.

All code is written with level four warnings and warnings as errors using Microsoft Visual C++ 6.0.
